342. 4的幂

给定一个整数，写一个函数来判断它是否是 4 的幂次方。如果是，返回 true ；否则，返回 false 。

整数 n 是 4 的幂次方需满足：存在整数 x 使得 n == $4^x$

思路

经观察，不难发现：

  1. 4的幂次方一定是2的幂次方
  2. 写成二进制后，可知4的幂的二进制表示中只会出现一位1，而且该1一定出现在奇数位上
  3. 在n为2的幂次方的前提下，若n除以3的余数为1，则为4的幂次方，否则不然
